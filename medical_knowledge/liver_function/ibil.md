# 间接胆红素（IBIL）

## 指标作用

反映尚未在肝内结合的胆红素部分。

## 升高可能原因

- 可能与红细胞破坏增多、胆红素结合能力变化或样本因素有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 通常无独立异常意义，需结合血常规和其他胆红素指标。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
