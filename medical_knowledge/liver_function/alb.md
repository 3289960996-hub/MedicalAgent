# 白蛋白（ALB）

## 指标作用

维持血浆胶体渗透压并运输多种物质，可反映蛋白合成与营养状态。

## 升高可能原因

- 可能与脱水、血液浓缩或个体差异有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 可能与摄入不足、合成减少、消耗增加、丢失增多或稀释有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
