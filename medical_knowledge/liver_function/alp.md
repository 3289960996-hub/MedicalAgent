# 碱性磷酸酶（ALP）

## 指标作用

主要来源于肝胆系统和骨组织，儿童生长期可生理性升高。

## 升高可能原因

- 可能与胆汁排出变化、骨代谢活跃、生长发育或妊娠有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 可能与营养状态、微量元素不足或个体差异有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
