# 丙氨酸氨基转移酶（ALT）

## 指标作用

反映肝细胞内酶释放情况，是评估肝细胞损伤的常用指标。

## 升高可能原因

- 可能与肝细胞受损、剧烈运动、肌肉损伤或样本因素有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 单独偏低通常临床意义有限，也可能与个体差异或营养状态有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
