# 胆碱酯酶（CHE）

## 指标作用

主要由肝脏合成，可辅助观察肝合成功能和营养状态。

## 升高可能原因

- 可能与代谢状态、肥胖、血脂变化或个体差异有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 可能与合成能力下降、营养不足或某些物质暴露有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
