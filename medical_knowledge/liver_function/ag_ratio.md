# 白蛋白/球蛋白比值（A/G）

## 指标作用

反映白蛋白与球蛋白的相对比例。

## 升高可能原因

- 可能与白蛋白相对增多、球蛋白减少或脱水有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 可能与白蛋白减少、球蛋白增多、蛋白丢失或慢性免疫刺激有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
