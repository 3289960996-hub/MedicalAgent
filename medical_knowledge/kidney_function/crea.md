# 肌酐（CREA）

## 指标作用

肌肉代谢产物，经肾脏滤过，用于辅助评估肾小球滤过。

## 升高可能原因

- 可能与滤过能力下降、脱水、肌肉量较大、剧烈运动或饮食有关。
- 结果还可能受采样时点、生理状态、饮食、运动、药物和实验室参考区间影响。

## 降低可能原因

- 可能与肌肉量较少、营养状态、妊娠或血液稀释有关。
- 应结合历史趋势、同组指标和本实验室参考范围复核。

## AI解释规范

允许：
- 使用“可能提示”等不确定性表述。
- 明确说明“需要结合其他指标判断”。
- 提醒结合症状、病史、采样条件和动态复查结果。

禁止：
- 直接诊断疾病。
- 推荐药物。
- 制定治疗方案。
